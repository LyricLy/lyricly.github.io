If you want to know how to get BootAnim9 set up, this is the place to be

Prerequisites:

  • For A9LH: BootAnim9
  • For B9S: BootCTR9
  • Boot Animation(s)

For A9LH users:

  1. Create a folder named anim on the root of your SD card
  2. Download and extract the contents of the BootAnim9.zip to the root of your SD card
    • When prompted, choose "Move, but keep both files"
  3. Rename arm9loaderhax (1).bin on the root of your SD card to arm9loaderhax_si.bin
  4. Move arm9loaderhax.bin on the root of your SD card to the anim folder
  5. Rename arm9loaderhax.bin in the anim folder to arm9payload.bin
  6. Create a folder named 0 in the anim folder and extract your boot animation to it

For B9S users:

  1. Create a folder named anim on the root of your SD card
  2. Rename boot.firm on the root of your SD card to luma.firm
  3. Move luma.firm to the boot9strap folder
  4. Download and extract the contents of BootCTR9.zip onto the root of your SD card
  5. Create a folder named 0 in the anim folder and extract your boot animation to it

If you are using A9LH and want to add another boot animation, create a 1 folder on root; for the next create a 2 folder and so on. This does not work for B9S users yet

Do not skip any numbers, folders must be created in order
